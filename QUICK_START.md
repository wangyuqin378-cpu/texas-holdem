# 🎮 快速开始 - 德州扑克(含机器人)

## 1️⃣ 启动游戏

```bash
cd texas-holdem
npm install  # 首次运行需要安装依赖
npm start    # 启动服务器
```

服务器会在 `http://localhost:3000` 启动

## 2️⃣ 打开游戏

在浏览器访问: `http://localhost:3000`

## 3️⃣ 添加机器人玩家

1. 输入你的昵称(比如"玉斧")
2. 点击"创建房间"或"快速加入"
3. 进入房间后,在底部看到**机器人控制按钮**:
   - 🤖 简单 - 适合新手
   - 🤖 中等 - 推荐选择  ⭐
   - 🤖 困难 - 高手对战

4. 点击任意难度按钮添加机器人
5. 可以添加多个机器人(最多6个,总共7人)
6. 点击"准备"按钮
7. 机器人会自动准备,2秒后游戏开始!

## 4️⃣ 游戏玩法

- **操作**: 弃牌 / 过牌 / 跟注 / 加注 / 全下
- **倒计时**: 每回合120秒(2分钟)
- **轮数**: 共20轮
- **重购**: 筹码用完可以重购1000

## 5️⃣ 机器人特性

✅ 自动操作 - 不需要等待
✅ 智能决策 - 基于牌力、赔率、位置
✅ 学习对手 - 会记录你的打法
✅ 模拟思考 - 有真实的思考延迟

### 三种难度对比

| 难度 | 诈唬频率 | 策略特点 | 推荐场景 |
|-----|---------|---------|---------|
| 简单 | 5% | 纯牌力决策 | 新手练习 |
| 中等 | 15% | 位置+对手分析 | 日常对战 ⭐ |
| 困难 | 20% | GTO策略+极化打法 | 高手挑战 |

## 6️⃣ 常见场景

### 单人练习
```
1. 创建房间
2. 添加6个中等机器人
3. 准备开始
```

### 朋友对战+机器人
```
1. 创建房间
2. 添加3个机器人
3. 分享房间号给朋友
4. 朋友输入房间号加入
5. 准备开始
```

### 移除机器人
```
点击"移除机器人"按钮清空所有AI
(只能在等待阶段移除)
```

## 7️⃣ 键盘快捷键

- **Enter** - 发送聊天消息
- **聊天窗** - 可拖动到任意位置

## 8️⃣ 常见问题

**Q: 机器人会作弊吗?**
A: 不会!机器人只能看到自己的手牌,和真人一样。

**Q: 可以调整机器人难度吗?**
A: 可以混合添加不同难度的机器人,比如2个简单+2个困难。

**Q: 机器人会超时吗?**
A: 不会,机器人会在思考后自动行动,不会触发超时。

**Q: 如何让游戏更快?**
A: 只添加简单机器人,它们的思考时间最短(800-1500ms)。

**Q: 20轮结束后怎么办?**
A: 查看结算排名后,点击"再来一场"重新开始。

## 9️⃣ 进阶玩法

### 测试你的策略
```
1. 添加6个困难机器人
2. 观察它们如何打牌
3. 学习GTO策略
```

### 混合难度挑战
```
简单×2 + 中等×2 + 困难×2
模拟真实桌子的多样性
```
