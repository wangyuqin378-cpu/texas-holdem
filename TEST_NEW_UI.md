# 🎮 新UI测试指南

## 改进内容

### ✅ 座位上的加号按钮
- 空座位显示大 `+` 按钮
- 点击 `+` 添加困难机器人

### ✅ 机器人座位的删除按钮  
- 机器人座位右上角显示 `×` 按钮
- 点击 `×` 移除该机器人

### ✅ 自动准备机制
- 真人点"准备"后
- 所有机器人自动准备
- 有2人准备即可开始

### ❌ 移除的内容
- 底部的机器人按钮（简单/中等/困难）
- 移除所有机器人按钮

## 使用流程

1. **启动服务器**
```bash
cd texas-holdem
npm start
```

2. **打开浏览器**
访问 `http://localhost:3000`

3. **创建房间**
输入昵称，点击"创建房间"

4. **添加机器人**
- 看到7个座位
- 你的座位在正下方
- 其他空座位显示大 `+` 按钮
- 点击任意 `+` 添加机器人

5. **准备开始**
- 点击底部"准备"按钮
- 机器人自动准备
- 2秒后游戏开始

6. **移除机器人**
- 机器人座位右上角有 `×` 按钮
- 点击即可移除
