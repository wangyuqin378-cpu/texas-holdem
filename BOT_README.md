# 🤖 德州扑克机器人功能说明

## 功能特性

现在你的德州扑克游戏支持添加**智能AI机器人**,即使一个人也可以畅玩!

### 三种难度等级

1. **🟢 简单 (Easy)**
   - 基于基础牌力和赔率决策
   - 诈唬频率: 5%
   - 适合新手练习基本玩法

2. **🟡 中等 (Medium)** ⭐ 推荐
   - 考虑位置、对手风格、适度诈唬
   - 诈唬频率: 15%
   - 会根据你的打法动态调整策略
   - 适合大多数玩家,有一定挑战性

3. **🔴 困难 (Hard)**
   - 简化版GTO(博弈论最优)策略
   - 诈唬频率: 20%
   - 极化打法、平衡价值下注与诈唬
   - 适合高手对战,非常有挑战性

## 使用方法

### 快速开始

1. **创建房间或加入房间**
   - 输入昵称后创建房间

2. **添加机器人**
   - 在准备界面点击对应难度的机器人按钮:
     - `🤖 简单` - 添加简单机器人
     - `🤖 中等` - 添加中等机器人  
     - `🤖 困难` - 添加困难机器人
   - 可以混合添加不同难度的机器人
   - 最多支持6个机器人(总人数7人)

3. **开始游戏**
   - 点击"准备"按钮
   - 机器人会自动准备
   - 2秒后游戏开始

4. **移除机器人**
   - 点击"移除机器人"按钮可清除所有机器人
   - 只能在等待阶段移除

### 机器人行为

- ✅ **自动操作**: 机器人会在轮到它时自动思考并行动
- ✅ **思考时间**: 模拟真实玩家,有800-2500ms的思考延迟
- ✅ **自动准备**: 添加机器人后会自动准备
- ✅ **自动确认**: 每轮结束后自动确认下一局
- ✅ **学习对手**: 会记录你的打法风格并调整策略

## 机器人决策系统

### 简单机器人
```
牌力 < 0.25  → 弃牌/过牌
牌力 0.25-0.5 → 跟注(如果赔率好)
牌力 0.5-0.7  → 跟注
牌力 > 0.7    → 加注/全下
```

### 中等机器人
- 考虑位置优势(晚位更激进)
- 分析对手激进程度
- 根据赔率和底池大小决策
- 适度诈唬(15%概率)
- 记忆对手打法并调整

### 困难机器人
- 实施GTO范围策略
- 极化打法(要么强牌价值下注,要么诈唬)
- 根据阶段动态调整范围
- 平衡慢打与激进
- 防守vs激进对手
- 高级位置利用

## 技术实现

### 后端架构
```
server/
├── botPlayer.js       # 机器人AI核心逻辑
├── botManager.js      # 机器人管理器
├── handEvaluator.js   # 牌力评估(新增getHandStrength)
└── index.js           # 集成机器人到游戏服务器
```

### 核心算法

1. **牌力评估**: 
   - 将牌型转换为0-1的强度值
   - 考虑kicker的影响

2. **赔率计算**:
   - `PotOdds = 当前底池 / (底池 + 跟注额)`

3. **位置价值**:
   - 庄家位置 = 1.0(最好)
   - 小盲位置 = 0(最差)
   - 线性分布

4. **对手画像**:
   - 记录每个对手的激进度、弃牌率
   - 使用加权平均更新(新观察权重递减)

### 调整机器人参数

编辑 `server/botPlayer.js` 中的 `BOT_CONFIGS`:

```javascript
const BOT_CONFIGS = {
  medium: {
    name: 'Bot中等',
    thinkTime: [1000, 2000],    // 思考时间范围
    bluffFrequency: 0.15,       // 诈唬频率
    aggressiveness: 0.5,        // 激进程度
    foldThreshold: 0.3,         // 弃牌阈值
    callThreshold: 0.55,        // 跟注阈值
    raiseThreshold: 0.75,       // 加注阈值
  }
}
```

## 常见问题

**Q: 机器人会作弊吗?**
A: 绝对不会。机器人只能看到自己的手牌,和真人玩家获得的信息完全一样。

**Q: 机器人会断线吗?**
A: 不会。机器人是虚拟玩家,永远在线。

**Q: 可以只和机器人玩吗?**
A: 可以!一个人也能玩,添加6个机器人即可7人局。

**Q: 机器人会重购吗?**
A: 当前版本机器人不会主动重购。20轮结束后可以重新开始。

**Q: 如何让游戏更有挑战?**
A: 添加多个困难机器人,它们会使用GTO策略并互相博弈。

**Q: 机器人的思考时间可以加快吗?**
A: 可以。修改 `thinkTime: [min, max]` 参数即可,单位是毫秒。
