## 🎯 机器人决策算法

### 简单机器人
```javascript
if (牌力 < 0.25) → 弃牌
if (牌力 < 0.5) → 跟注(赔率好时)
if (牌力 >= 0.7) → 加注/全下
```

### 中等机器人
```javascript
调整牌力 = 基础牌力 + 位置加成
if (应该诈唬 && 对手弱) → 加注
if (对手激进) → 提高弃牌阈值
根据赔率和底池决策
```

### 困难机器人
```javascript
使用GTO范围策略
极化打法(强牌/诈唬,避免中牌)
平衡价值下注与诈唬比例
防守vs激进对手
动态调整每个阶段的范围
```

## 🔧 参数调优建议

### 提高难度
```javascript
// botPlayer.js
medium: {
  bluffFrequency: 0.20,  // 提高诈唬
  aggressiveness: 0.7,   // 更激进
  thinkTime: [500, 1000] // 更快决策
}
```

### 降低难度
```javascript
easy: {
  bluffFrequency: 0.02,  // 几乎不诈唬
  aggressiveness: 0.2,   // 保守
  foldThreshold: 0.3     // 容易弃牌
}
```
